Sobre o Projeto PAP

Projeto de Aptidão Profissional

Este site foi desenvolvido como parte do meu Projeto de Aptidão Profissional (PAP).

O objetivo deste projeto é criar um guia interativo sobre os jogos Hollow Knight e Hollow Knight: Silksong, apresentando informações detalhadas sobre mecânicas de jogo, itens e inimigos do jogo muito mais.

Através deste trabalho, procurei demonstrar as competências técnicas adquiridas ao longo do curso, incluindo desenvolvimento web, design de interface e organização de conteúdo.
